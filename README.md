## 🚀 Quick Start (2 minutes)

### **Prerequisites**
- Node.js v16+ | [Get it here](https://nodejs.org/)
- npm/yarn | *Included with Node.js*

### **Setup**

```bash
# Clone the repository
git clone https://github.com/anjalibirudu/LEAFWISE-GUARDIAN
cd LEAFWISE-GUARDIAN

# Install & run
npm install && npm run dev
```

✅ **App runs at** `http://localhost:5173`

## 📁 Project Architecture

```
src/
├── 📦 components/      Reusable, tested React components
├── 📄 pages/           Page-level components with routing
├── 🎣 hooks/           Custom React hooks for logic
├── 🛠️  utils/           Helper functions & utilities
├── 📝 types/           TypeScript definitions (type-safe)
└── 🎨 App.tsx          Main application entry
```

**Architecture Highlights:**
- ✅ Single Responsibility Principle
- ✅ DRY (Don't Repeat Yourself)
- ✅ Component reusability & composability
- ✅ Clear separation of concerns
